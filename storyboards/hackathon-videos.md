# Hackathon Videos — Storyboard

## Segment 1 (0:00–0:09)

### Video Description
Three split screens, each showing a different hackathon. Countdown timers ticking. Teams coding furiously. Whiteboards covered in architecture diagrams. Three project names flash: r/factchecker — a misinformation detection tool. RecruitBolt — an AI-powered recruitment platform. SpecSync — a specification synchronisation engine.

### Script
"Three hackathons. Three completely different problems. r/factchecker fights misinformation. RecruitBolt speeds up hiring. SpecSync keeps technical specs in sync. All built under pressure."

## Segment 2 (0:09–0:18)

### Video Description
The code is done. But the deadline is in hours and there's no demo video. At a hackathon, judges don't read code — they watch the pitch. Miguel opens a video editor. Screen recordings drop in. He writes a script. Amazon Polly generates the narration — clear, professional voice. He cuts, arranges, polishes. The clock is ticking.

### Script
"The code was finished. But at a hackathon, judges don't read your code — they watch your pitch. With hours left, Miguel became the filmmaker. Script. Narration. Edit."

## Segment 3 (0:18–0:27)

### Video Description
Three polished demo videos play side by side — each one clean, professional, telling a compelling story. Devpost submission pages load. Submit buttons click — one, two, three. All before the deadline. View counts start ticking up. Comments arrive. The projects weren't just built — they were sold.

### Script
"Three demos. Three submissions. All delivered before the deadline. The projects didn't just work — the videos made people believe in them."
