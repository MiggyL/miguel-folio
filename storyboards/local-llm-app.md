# Local LLM App — Storyboard

## Segment 1 (0:00–0:09)

### Video Description
A boardroom. Executives around a table discussing strategy. Documents stamped "CONFIDENTIAL" on the table. One person asks: "Can we use AI to help with this?" Another pulls up ChatGPT — then pauses. Looks at the confidential documents. Closes the browser tab. Shakes his head.

### Script
"They needed AI to process sensitive documents. But every AI tool sends data to the cloud. And this data — contracts, strategies, client names — can never leave the building."

---

## Segment 2 (0:09–0:18)

### Video Description
A single laptop on the boardroom table. Someone opens it. A clean chat interface loads — no browser, no cloud icon, no loading from external servers. A network activity monitor in the corner shows zero outbound connections. The AI is already running. Locally. On this machine.

### Script
"So the AI came to them. A full language model running on a single laptop. No cloud. No API calls. No data leaves the machine. Not even a ping."

---

## Segment 3 (0:18–0:27)

### Video Description
The executive who closed the browser earlier now types a question into the local AI. The response generates word by word — fast, accurate. He asks a follow-up about a client contract. It answers with context. He looks up at the room. Nods. Others start gathering around the laptop.

### Script
"He asked about the contract. The AI answered — with context, with nuance. No data leaked. No terms of service violated. Just intelligence, completely private."

---

## Segment 4 (0:27–0:36)

### Video Description
The boardroom empties. The laptop closes. The confidential documents go back in the safe. But the AI helped — decisions were made, documents were analysed, questions were answered. All without a single byte leaving the room. Camera pulls back through the office. Fade to black.

### Script
"Every question answered. Every document analysed. And when the laptop closed, not a single byte had left the room. AI without compromise."
